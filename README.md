# Weather app

Simple weather app made with Visual Crossing API (https://www.visualcrossing.com/)

The app provides all weather information.

## Functions

-  search for any city or country
-  hourly forecast
-  forecast for the next 15 days
-  light / dark mode
-  EU / US unit converter

## Build using:

-  webpack
-  HTML
-  CSS
-  JS ES6 modules

## What I learned

-  connecting with API
-  asynchronous code
